# Portfolio Starter (Static, No-Build)

Einfaches, schnelles Portfolio für **GitHub Pages**. Keine Builds, kein Jekyll erforderlich.

## Los geht's
1. Neues Repository (z. B. privat) erstellen.
2. Inhalt dieses Ordners hochladen.
3. Lokal testen: `index.html` doppelklicken.
4. Wenn alles passt: in ein öffentliches Repo **<username>.github.io** kopieren.

## Anpassen
- `Mecki` + E-Mail ersetzen.
- CV unter `assets/Mecki_CV.pdf`.
- Bilder in `assets/images/` austauschen.
- Optional: `.nojekyll` belassen (empfohlen).

Viel Erfolg!
